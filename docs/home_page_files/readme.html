Игра-лабиринт

О проекте

Данный проект - простенький лабиринт, написанный на языке python с помощью библиотеки tkinter.

Введение

Добро пожаловать в игру "Где Прога?"! Сюжет данной игры предельно прост: Вы играете за восьмиклассника Сашу, который совсем недавно поступил в ФТШ. У нашего героя через 15 минут начинается первый урок в его новой школе - информатика, Однако как это часто бывает с новенькими, он не может найти нужный ему кабинет. Ваша задача - помочь Саше найти путь к кабинету информатики и не получить по лбу от учителя.
